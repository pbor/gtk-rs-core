fn next<P: IsA<Cancellable>>(
    &self,
    cancellable: Option<&P>
) -> Result<SocketAddress, Error>[src]

Retrieves the next crate::SocketAddress from self. Note that this may block for some amount of time. (Eg, a crate::NetworkAddress may need to do a DNS lookup before it can return an address.) Use Self::next_async() if you need to avoid blocking.

If self is expected to yield addresses, but for some reason is unable to (eg, because of a DNS error), then the first call to Self::next() will return an appropriate error in *error. However, if the first call to Self::next() succeeds, then any further internal errors (other than cancellable being triggered) will be ignored.

cancellable

optional crate::Cancellable object, None to ignore.

Returns

a crate::SocketAddress (owned by the caller), or None on error (in which case *error will be set) or if there are no more addresses.

Asynchronously retrieves the next crate::SocketAddress from self and then calls callback, which must call [Self::next_finish()] to get the result.

It is an error to call this multiple times before the previous callback has finished.
