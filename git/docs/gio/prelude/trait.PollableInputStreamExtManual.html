Creates a crate::glib::Source that triggers when self can be read, or cancellable is triggered or an error occurs. The callback on the source is of the GPollableSourceFunc type.

As with [crate::PollableInputStream::is_readable()], it is possible that the stream may not actually be readable even after the source triggers, so you should use [crate::PollableInputStream::read_nonblocking()] rather than [crate::InputStream::read()] from the callback.

Returns

fn read_nonblocking<C: IsA<Cancellable>>(
    &self,
    buffer: &mut [u8],
    cancellable: Option<&C>
) -> Result<isize, Error>[src]

Attempts to read up to count bytes from self into buffer, as with [crate::InputStream::read()]. If self is not currently readable, this will immediately return crate::IOErrorEnum::WouldBlock, and you can use [crate::PollableInputStream::create_source()] to create a crate::glib::Source that will be triggered when self is readable.

Note that since this method never blocks, you cannot actually use cancellable to cancel it. However, it will return an error if cancellable has already been cancelled when you call, which may happen if you call this method after a source triggers due to having been cancelled.

buffer

a buffer to read data into (which should be at least count bytes long).

count

the number of bytes you want to read

cancellable

a crate::Cancellable, or None

Returns

the number of bytes read, or -1 on error (including crate::IOErrorEnum::WouldBlock).
