fn query_info<P: IsA<Cancellable>>(
    &self,
    attributes: &str,
    cancellable: Option<&P>
) -> Result<FileInfo, Error>[src]

Queries a file io stream for the given attributes. This function blocks while querying the stream. For the asynchronous version of this function, see Self::query_info_async(). While the stream is blocked, the stream will set the pending flag internally, and any other operations on the stream will fail with crate::IOErrorEnum::Pending.

Can fail if the stream was already closed (with error being set to crate::IOErrorEnum::Closed), the stream has pending operations (with error being set to crate::IOErrorEnum::Pending), or if querying info is not supported for the stream’s interface (with error being set to crate::IOErrorEnum::NotSupported). I all cases of failure, None will be returned.

If cancellable is not None, then the operation can be cancelled by triggering the cancellable object from another thread. If the operation was cancelled, the error crate::IOErrorEnum::Cancelled will be set, and None will be returned.

attributes

a file attribute query string.

cancellable

optional crate::Cancellable object, None to ignore.

Returns

a crate::FileInfo for the self, or None on error.
